# 길드는 잠들지 않는다 — 핵심 기획서 (현재 구현 기준)

> 이 문서는 현재 구현된 게임 상태를 기준으로 작성되었습니다.
> 확장 콘텐츠(4~5차 전직, 미구현 항목)는 `2_확장_기획서.md`를 참고하세요.
> 직업 변경 내역은 `3_직업_변경사항.md`를 참고하세요.

---

## 1. 기본 정보

- **장르**: 텍스트 기반 방치형 게임
- **플랫폼**: 웹브라우저 (반응형)
- **배경**: 용의 시대 세계관, 봉건국 베른
- **플레이어 역할**: 베른 모험가 길드장
- **UI 톤**: 밝은 판타지풍 색감, 큼직한 버튼, 높은 가시성

---

## 2. 핵심 게임 루프

```
모험가 모집 → 팀 구성 → 파견 → 정산(골드+재료+장비) → 인프라 업그레이드 → 더 높은 지역 파견
```

---

## 3. 재화 시스템

| 재화 | 획득처 | 사용처 |
|------|--------|--------|
| 골드 | 파견 정산 | 모험가 모집, 서류 갱신, 건물 건설, 영구 단련 |
| 재료 | 파견 정산 | 건물 업그레이드, 경험치 책 제작 |
| 장비 | 파견 정산 | 모험가 장착 → 능력치 강화 |

> 재료는 현재 단일 종류(숫자)로만 관리됩니다. 등급 구분 없음.

## 4. 모험가 시스템

### 모집 방식
- 클로에가 일정 시간마다 모험가 서류 3장을 가져옴
- 원하는 모험가를 골라 길드에 합류시킴 (3장 중 복수 선택 가능)
- 15분마다 서류 전체 자동 갱신
- 골드를 내면 즉시 수동 갱신 가능 (비용 점진 증가)

### 기본 능력치
- 체력 / 공격력 / 방어력 / 속도 / 치명타 확률 / 치명타 데미지

### 특성 시스템
- 모험가마다 특성 1~2개 랜덤 보유 (40% 확률로 2개)
- 특성은 전투 능력치(체력, 공격력, 방어력 등)에만 영향

### 레벨업
- 연구소에서 제작 또는 상점에서 구매한 경험치 책을 사용해 레벨업
- 파견 전투 중 경험치도 획득 (단계 비례 증가)

### 직업 트리 및 스킬 (1~3차)

#### 전사 계열 (버티는 스타일)
```
전사 (기본 스킬: 강타)
├── 기사 (도발 태세 / 방패 강타) → 수호기사 (성벽 / 심판의 일격)
└── 검투사 (반격 태세 / 회오리) → 챔피언 (끓어오르는 피 / 폭풍)
```

#### 도적 계열 (빠르고 치명타 위주)
```
도적 (기본 스킬: 기습)
├── 암살자 (급소 노리기 / 독침) → 닌자 (대규모 암살 / 독안개)
└── 사냥꾼 (표식 / 연사) → 현상금 사냥꾼 (집착 / 확인 사살)
```

#### 마법사 계열
```
마법사 (기본 스킬: 마력 난사)
├── 대성당 술사 (수호 술식 / 접근 거부) → 대성당 학자 (수호 술식 2.0 / 시스템 격리)
└── 상아탑 술사 (좌표 타격 / 연발 번개) → 상아탑 현자 (좌표 붕괴 / 번개 지옥)
```

> 대성당 계열은 회복 없음. 보호막 + 적 공격력 디버프(접근 거부/시스템 격리) 중심.

#### 치유사 계열 (회복 특화, 신규 추가)
```
치유사 (기본 스킬: 응급 처치, 기본 공격이 가장 HP 낮은 아군 회복으로 치환)
├── 성직자 (빛의 장막/패시브 / 따스한 빛/광역 회복) → 신관 (빛의 축복 / 포옹하는 빛)
└── 용사제 (처벌/단일 공격 / 설교/전체 공격) → 심문관 (단죄/흡혈 / 웅변/명중 디버프)
```

- 속도 스탯이 전 직업 중 최하위 (기본값 × 0.5)
- 기본 공격이 "가장 HP 낮은 아군 회복 (ATK × 1.0)"으로 치환
- 마법사와 지팡이(무기 슬롯) 공유

### 전직 조건
- 2차 전직: Lv.10 + 골드 2,000 + 재료 10
- 3차 전직: Lv.25 + 골드 8,000 + 재료 30 + **경지 개방 Ⅱ 필요** (경지 탭에서 개방, 5 스킬포인트 소요)

## 5. 파견 시스템

### 기본 구조
- 팀 구성: 1~3명 자유 구성
- 파견지 하나당 팀 하나만 입장 가능
- 동시 파견 팀 수: 지휘 본부 레벨로 결정
- 인원 교체: 귀환 후에만 가능

### 파견 진행
- 파견 후 모험가는 귀환 버튼을 누를 때까지 계속 활동
- 골드 + 재료 + 장비가 지속적으로 누적
- 플레이어가 원할 때 정산 버튼으로 전부 회수

### 던전 공략 진행도
- 전멸 시 **기본 진행도 50% 보존** (리빌딩 경지 노드로 최대 75%까지 상승)
- 진행도가 높을수록 강한 몬스터, 더 좋은 장비 드롭

### 파견 지역 구조 (총 40단계)

| 국가 | 단계 | 특징 |
|------|------|------|
| 봉건국 베른 | 1~5 | 초보 구간, 동물·곤충류 |
| 풀무불의 칸 | 6~10 | 기계·광산 몬스터 |
| 무역 연합 솔 | 11~15 | 해양·수중 몬스터 |
| 교국 에테리아 | 16~20 | 성스러운 몬스터 |
| 신성 제국 오르도스 | 21~25 | 군사·전쟁 몬스터 |
| 기계 공화국 아이언 피스트 | 26~30 | 기계·자동화 몬스터 |
| 드래곤 왕국 | 31~35 | 고위 마물·드래곤 |
| 미지의 영역 | 36~40 | 최강 몬스터, 보스 티아마트 |

## 6. 전투 시스템

### 기본 구조
- 최대 3vs3 (아군 최대 3명, 적 최대 3마리)
- 보스: 무조건 1마리, 진행도 기반 출현율
- 속도 스탯 높은 순서대로 행동
- **6초마다 전투 1회** (파견 중 자동 진행)
- 탐색 30% / 전투 70% 비율

### 체력 및 전멸
- 전멸 시 전원 체력 풀 회복 후 자동 재도전
- 전멸 시 진행도 **50% 보존** (기본값, 경지 노드로 최대 75%)
- 전멸 없이 전투 승리 시 남은 체력 그대로 다음 전투

### 상태이상 목록
| 상태이상 | 효과 |
|---|---|
| 기절 (stun) | N턴간 행동 불가 |
| 독 (poison) | N턴간 매 턴 고정 피해 |
| 출혈 (bleed) | N턴간 매 턴 고정 피해 |
| 명중 하락 (acc_down) | 35% 확률 공격 빗나감 |
| 방어 감소 (def_break) | 방어력 40%로 감소 |
| 공격력 하락 (atk_down) | N턴간 공격력 X% 감소 |

## 7. 장비 시스템

### 장비 슬롯
- 무기 (1개) / 방어구 (1개) / 악세서리 (1개)
- 무기는 직업 계열(전사/도적/마법사·치유사) 전용

### 장비 등급
```
일반 / 마법 / 희귀 / 영웅 / 전설 / 신화 (6등급)
```

### 추가 옵션 수
- 일반: 0개 / 마법: 1개 / 희귀: 1개 / 영웅: 2개 / 전설: 2개 / 신화: 3개

## 8. 길드 인프라

| 건물 | 기능 | 업그레이드 효과 | 최대 레벨 |
|------|------|----------------|-----------|
| 지휘 본부 | 동시 파견 팀 수 | 레벨 = 파견 슬롯 수 | **10** |
| 대기실 | 보유 모험가 최대 수 | 레벨당 슬롯 +1 | **10** |
| 창고 | 장비 보관 | 레벨당 칸 +1 | **18** |
| 연구소 | 경험치 책 제작 + 영구 단련 | 더 좋은 책 제작, 시간 단축 | **12** |
| 모집 서류 | 모집 서류 질 향상 | 더 좋은 모험가 출현 | **8** |

### 연구소 세부 기능
- 골드 + 재료 투입 → 시간 경과 후 경험치 책 완성 (5종)
- **영구 단련**: 골드를 소모해 모험가 스탯을 영구적으로 강화 (리빌딩 후에도 유지)
  - 체력 단련: 300,000G 시작 / 공격 훈련: 400,000G 시작 / 방어 수련: 350,000G 시작
  - 레벨당 +1%, ×1.35 비용 증가

> 재료 합성 기능은 미구현 상태입니다. (재료가 단일 종류이므로 구조 변경 필요)

## 12. 리빌딩(프레스티지) 시스템

- 누적 골드 기반 스킬포인트 획득: `floor(sqrt(lifetimeGold / 25000))`
- 전체 트리 해금에 약 60 스킬포인트 필요 (~90M 골드)

### 경지 트리 구조

```
                    [경지 개방 Ⅰ / genesis]
          /              |              |              \
   [성장 가지]      [모험 가지]    [생존 가지]      [경지 가지]
   경험치/재화       진행도 보너스   HP/방어 강화    경지 개방 Ⅱ (5SP)
   획득량 증가                                       └─ 경지 개방 Ⅲ (12SP)
```

### 리빌딩 자동 보너스
- 리빌딩 횟수 × 3%를 HP/ATK/DEF 전체에 자동 적용

### 유지되는 데이터
- 누적 골드 / 스킬포인트 / 경지 노드 / 리빌딩 횟수 / 영구 단련 레벨
