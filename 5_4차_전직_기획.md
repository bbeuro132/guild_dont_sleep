# 길드는 잠들지 않는다 — 4차 전직 기획

> 3차 전직 스킬 2개를 유지하고, 패시브 스킬 1개를 추가하는 방식으로 구현합니다.
> 스킬 발동 우선순위: 쿨타임이 찬 스킬 중 번호 순서대로 (1번 → 2번 → 3번)
> 4차 전직 조건: 경지 트리 "경지 가지"의 경지 개방 II 노드 필요

---

## 전사 계열

### 성채 (수호기사 4차)

```
스킬1 - 성벽 (3차 유지)
  : CD 4, 6턴 도발 + 아군 전체 피해 경감 20%
  : 쿨타임은 지속 시간이 끝난 후부터 계산

스킬2 - 심판의 일격 (3차 유지)
  : CD 2, 단일 물리 ATK×1.7 + 기절 2턴 (보스 면역)

스킬3 - 축성 (패시브, 신규)
  : 기본 피해 경감 20% (전투 내내 항상 적용)
  : 성벽의 피해 경감 20%와 합연산 → 성벽 발동 시 최대 40% 경감
```

---

### 파괴자 (챔피언 4차)

```
스킬1 - 끓어오르는 피 (3차 유지)
  : CD 4, 4~5턴간 피해 경감 20% + 아군 피격 시마다 반격
  : 쿨타임은 지속 시간이 끝난 후부터 계산

스킬2 - 폭풍 (3차 유지)
  : CD 3, 적 전체 물리 ATK×1.5, 치명타 적용 가능

스킬3 - 약점 파악 (패시브, 신규)
  : 치명타 확률 +20% (전투 내내 항상 적용, 합연산)
```

## 도적 계열

### 대닌자 (닌자 4차)

```
스킬1 - 대규모 암살 (3차 유지)
  : CD 3, 치명타 확정 + 대상 사망 시 무작위 대상에게 연쇄 시전

스킬2 - 독안개 (3차 유지)
  : CD 4, 적 전체 3턴 독 (ATK×0.25/턴) + 1턴 명중률 감소

스킬3 - 기민함 (패시브, 신규)
  : 회피율 +25% (전투 내내 항상 적용, 합연산)
```

---

### 추적자 (현상금 사냥꾼 4차)

```
스킬1 - 집착 (3차 유지)
  : CD 4, 적 1명 집착 지정, 해당 대상에게 피해 +30%

스킬2 - 확인 사살 (3차 유지)
  : CD 3, 집착 대상에게 3연타 물리 (ATK×1.0×1.3/회)

스킬3 - 개를 풀어라 (패시브, 신규)
  : 기본 공격 시 공격력 30% 추가 피해 1타 (사냥개)
  : 기본 공격에만 적용 (스킬 공격 제외)
  : 전투 로그: "사냥개가 OOO를 물었다" 형태로 출력
```

## 마법사 계열

### 통제관 (대성당 학자 4차)

```
스킬1 - 수호 술식 2.0 (3차 유지)
  : CD 4, 아군 전체 강화 보호막 (ATK×3.5), 파괴 시 50% 반사

스킬2 - 시스템 격리 (3차 유지)
  : CD 3, 적 전체 마법 ATK×0.4 + 3턴 공격력 35% 하락

스킬3 - 제어 프로토콜 (패시브, 신규)
  : 전투 시작 시 적 전체 속도 -30% (전투 내내 유지)
  : 합연산 적용
```

---

### 대현자 (상아탑 현자 4차)

```
스킬1 - 좌표 붕괴 (3차 유지)
  : CD 3, 단일 마법 ATK×2.8, 보호막·방어·회피 모두 무시

스킬2 - 번개 지옥 (3차 유지)
  : CD 3, 5~7회 마법 공격 (ATK×0.85/회), 랜덤 대상

스킬3 - 마력 담금질 (패시브, 신규)
  : 마법 공격력 +10% (전투 내내 항상 적용, 합연산)
  : 좌표 붕괴, 번개 지옥 모두 적용
```

## 치유사 계열

### 빛의 사도 (신관 4차)

```
스킬1 - 빛의 축복 (3차 유지, 패시브)
  : 피격 가중치 25%로 대폭 감소 (전투 내내 적이 거의 안 노림)

스킬2 - 포옹하는 빛 (3차 유지)
  : CD 5 (3회 소모 후 계산 시작)
  : 다음 3번의 기본 공격을 아군 전체 회복 (ATK×0.6)으로 전환

스킬3 - 헌신 (패시브, 신규)
  : 아군 전체가 받는 회복량 +10% (전투 내내 항상 적용)
  : 신관의 회복뿐 아니라 모든 회복 효과(흡혈, 장비 옵션 등)에 적용
```

---

### 광신자 (심문관 4차)

```
스킬1 - 단죄 (3차 유지)
  : CD 3, 단일 마법 ATK×2.2 + 피해량의 40%를 HP 최저 아군 회복

스킬2 - 웅변 (3차 유지)
  : CD 3, 적 전체 마법 ATK×0.6 + 1턴 명중률 감소

스킬3 - 용의 축복 (패시브, 신규)
  : 아군의 턴이 돌아올 때마다 해당 아군의 체력을 광신자 공격력의 10%만큼 회복
  : 광신자 공격력이 높아질수록 파티 전체 재생량 증가
```

## 구현 시 참고사항

- 모든 패시브는 전투 시작 시 자동 적용, 별도 발동 없음
- 수치 합연산 적용 (곱연산 아님)
- 통제관의 제어 프로토콜(속도 -30%)은 전투 시작 시 1회 적용, 전투 내내 유지
- 추적자의 개를 풀어라는 기본 공격에만 적용 (집착, 확인 사살 등 스킬 공격 제외)
- 광신자의 용의 축복은 "아군의 턴"마다 발동 (적의 턴 제외)
- 빛의 사도의 헌신은 모든 종류의 회복에 적용 (흡혈, 장비 옵션 포함)
- 4차 전직 조건 수치는 구현 시 Claude Code와 밸런스 협의
