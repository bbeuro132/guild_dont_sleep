## 1. 대성당 술사 계열 변경 ✅ 구현 완료

**컨셉**: 회복 기능 전부 제거 → **보호 + 통제(디버프) 중심 서브딜러**

### 2차 - 대성당 술사
```
스킬1 - 수호 술식 (battle_start)
  : 파티 전체 보호막 부여 (ATK × 2)
  : 보호막 파괴 시 타격한 적에게 × 0.5 피해 반사

스킬2 - 접근 거부 (cooldown 3)
  : 적 전체 소량 피해 (마법, ATK × 0.4)
  : 2턴간 적 전체 공격력 25% 하락 (atk_down 상태이상)
```

### 3차 - 대성당 학자
```
스킬1 - 수호 술식 2.0 (battle_start)
  : 보호막 양 증가 (ATK × 3.5)

스킬2 - 시스템 격리 (cooldown 3)
  : 접근 거부 강화
  : 적 전체 소량 피해 (마법, ATK × 0.4)
  : 3턴간 적 전체 공격력 35% 하락
```

## 2. 신규 직업군: 치유사 계열 ✅ 구현 완료

### 공통 특성
- **무기 슬롯**: 지팡이 (마법사와 동일 슬롯 공유)
- **속도**: 기본값 × 0.5 (전 직업 중 최하위)
- **기본 공격 메커니즘**: 공격력 × 1.0만큼 **가장 HP 낮은 아군 회복**

### 1차 - 치유사
```
기본 스킬 - 응급 처치 (cooldown 4, 전직 시 사라짐)
  : 가장 HP 낮은 파티원 대량 회복 (ATK × 3.0)
```

### 2차A - 성직자
```
스킬1 - 빛의 장막 (battle_start, 패시브)
  : 피격 가중치 50%로 감소 (기본 1 → 0.5)

스킬2 - 따스한 빛 (cooldown 3)
  : 파티원 전체 소량 회복 (ATK × 0.8)
```

### 2차B - 용사제
```
스킬1 - 처벌 (cooldown 3)
  : 적 1명에게 마법 피해 (ATK × 1.8)

스킬2 - 설교 (cooldown 3)
  : 적 전체에게 마법 피해 (ATK × 0.6)
```

### 3차A - 신관 (성직자 계열)
```
스킬1 - 빛의 축복 (passive)
  : 피격 가중치 25%로 감소 (빛의 장막보다 강화)

스킬2 - 포옹하는 빛 (cooldown 5, 3턴 소모 후 쿨타임 시작)
  : 3턴간 기본 공격이 파티 전체 소량 회복 (ATK × 0.6)으로 변경
  : 3턴을 모두 소모한 후부터 쿨타임 계산 시작
```

### 3차B - 심문관 (용사제 계열)
```
스킬1 - 단죄 (cooldown 3)
  : 적 1명에게 마법 피해 (ATK × 2.2)
  : 피해량의 40%만큼 가장 HP 낮은 아군 치유

스킬2 - 웅변 (cooldown 3)
  : 적 전체에게 마법 피해 (ATK × 0.6)
  : 1턴간 적 전체 명중률 하락 (acc_down)
```

## 3. 전체 직업 트리 (최종)

```
전사
├── 기사 → 수호기사
└── 검투사 → 챔피언

도적
├── 암살자 → 닌자
└── 사냥꾼 → 현상금 사냥꾼

마법사
├── 대성당 술사 (보호+통제, 회복 없음) → 대성당 학자
└── 상아탑 술사 (공격 특화) → 상아탑 현자

치유사
├── 성직자 (완전 치유 특화) → 신관
└── 용사제 (전투+치유 병행) → 심문관
```

## 5. 스킬 컨셉 수정 이력

### 기사 - 도발 태세
- 수정 전: 강제 타겟팅 (적이 반드시 자신을 공격)
- 수정 후: 피격 확률 대폭 상승 (가중치 기반 확률)
- 추가: 지속 시간(4턴) 종료 후부터 쿨타임(3턴) 시작

### 수호기사 - 성벽
- 수정 전: 강제 도발 + 아군 피해 경감 20%
- 수정 후: 피격 확률 대폭 상승 + 아군 피해 경감 20%
- 추가: 지속 시간(6턴) 종료 후부터 쿨타임(4턴) 시작

### 챔피언 - 끓어오르는 피
- 수정 전: 전투 전체 지속 (사실상 영구)
- 수정 후: 4~5턴 지속 (랜덤)
- 추가: 지속 시간 종료 후부터 쿨타임(4턴) 시작

### 닌자 - 대규모 암살
- 수정 전: 치명타 확정 + 다음 일반 공격이 적 전체 연쇄 공격으로 전환
- 수정 후: 치명타 확정 단일 공격 (ATK×1.9). 대상 사망 시 무작위 적에게 연쇄 시전

### 신관 - 포옹하는 빛
- 효과 변경 없음 (3턴간 기본 공격 → 전체 회복)
- 추가: 3턴을 모두 소모한 후부터 쿨타임(5턴) 시작
