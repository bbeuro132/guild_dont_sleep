## 2. 4차, 5차 전직 (미구현)

### 확장 방향
- 기존 2~3차 전직 패턴 유지: **스킬 강화 + 컨셉 심화**
- 스킬 개수는 2개로 유지
- 직업명, 스킬 내용은 추후 협의 필요

### 전직 트리 확장 구조
```
전사    → 기사 → 수호기사 → (4차) → (5차)
전사    → 검투사 → 챔피언 → (4차) → (5차)
도적    → 암살자 → 닌자 → (4차) → (5차)
도적    → 사냥꾼 → 현상금 사냥꾼 → (4차) → (5차)
마법사  → 대성당 술사 → 대성당 학자 → (4차) → (5차)
마법사  → 상아탑 술사 → 상아탑 현자 → (4차) → (5차)
치유사  → 성직자 → 신관 → (4차) → (5차)
치유사  → 용사제 → 심문관 → (4차) → (5차)
```

### 전직 조건 (현재 구현 기준)
```
2차 전직 → Lv.10 + 골드 2,000 + 재료 10 (제한 없음)
3차 전직 → Lv.25 + 골드 8,000 + 재료 30 + 경지 개방 Ⅱ 필요
4차 전직 → 미구현 (경지 개방 Ⅲ 연동 예정)
5차 전직 → 미구현
```

## 3. 프레스티지(리빌딩) 시스템 (부분 구현)

### 구현 완료
- 리빌딩 자체 (모험가/장비/골드/재료/건물 초기화, 영구 데이터 유지)
- 스킬포인트 획득: `floor(sqrt(lifetimeGold / 25000))`
- 경지 트리 14노드 (genesis / 성장 / 모험 / 생존 / 경지 가지)
- 리빌딩 횟수 자동 보너스: 횟수 × 3% (HP/ATK/DEF)
- 영구 단련 (연구소에서 별도 관리)

### 미구현 항목

**노드 취소 불가**: 모든 경지 노드는 구매 후 취소 불가. **이는 의도된 설계입니다.** (원 기획의 재배치 가능 안은 폐기)

**4차/5차 전직 개방**: 경지 개방 Ⅲ 노드에 4차 전직 잠금 해제 효과 추가 예정.

## 4. 재료 합성 (미구현, 재검토 필요)

원 기획: 연구소에서 낮은 등급 재료 여러 개 → 높은 등급 재료 1개 합성.

**현재 상태**: 재료가 단일 숫자(`State.materials`)로 관리되어 등급 구분 없음. 합성 기능 구현을 위해서는 재료 시스템 자체를 등급별로 분리하는 리팩터링이 선행되어야 함. 현재 우선순위에서 제외.

## 5. 상점 NPC (미반영)

황금의 길 마법사 NPC (밝고 사교적인 성격) 컨셉이 기획서에 있으나, 현재 상점은 단순 목록 UI로만 구현됨. NPC 소개 텍스트 / 대사 / 캐릭터 이미지 추가 예정.
